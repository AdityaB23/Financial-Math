
# Machine Learning Statistical Arbitrage (ETF Time Series)

This project implements a full end-to-end machine learning workflow for statistical arbitrage
using ETF time-series data. The pipeline includes:

- Data ingestion
- Feature engineering
- Predictive modeling
- Walk-forward validation
- Backtesting
- Performance evaluation

## Assets
SPY, QQQ, IWM, TLT, IEF, SHY, GLD

## Models
- Ridge Regression
- Random Forest
- XGBoost (optional)

## Features
- Lagged returns
- Rolling volatility
- Momentum
- Cross-asset spreads
- Z-score normalization

## Evaluation
- Sharpe ratio
- Drawdown
- Hit rate
- Cumulative returns

## Usage

1. Install requirements
2. Run notebooks in order
3. Inspect results in /results
